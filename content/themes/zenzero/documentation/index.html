1. Basic configuration

If you are happy with the web page as we organized it, there are only few steps you need to do before going online.

1.1 - Connect your Disqus account

For comments, Ghost uses external services such as Disqus. Zenzero already contains the necessary code to embed Disqus comments, you only need to change the account.
Go to assets/js/postScript.js on line 13 and insert your site short name of your, after you have registered.

FILE assets/js/postScript.js - line 13
var disqus_shortname = '';

1.2 - Choose your skin

You can use Zenzero in its basic version (purple) or one of the five alternative versions (blue, water, orange, gold, black), you just need to add the reference to its css.
Open default.hbs at line 17 and remove the comment from one of the link tag.

1.3 - Choose your animations

In addition to changing the skin, Zenzero includes several css3 animations that you can use on the different elements present in theme.
All you need to do is use the data attribute data-animate to assign an animation to any element.
The animations available are:

  • fadeIn
  • fadeDown
  • fadeUp
  • fadeLeft
  • fadeRight
  • fadeScale

1.5 - Edit the menu

Ghost 0.4.~ supports static pages, but does not provide a system to manage the menu.
For this reason you will need to edit the menu directly in the theme:
Simply open the file partials/menu.hbs and add the items that you need to ul, using as href the URL to the page (taken from the Ghost admin panel).
